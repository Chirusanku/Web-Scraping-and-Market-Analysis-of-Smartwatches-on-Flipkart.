# Web-Scraping-and-Market-Analysis-of-Smartwatches-on-Flipkart.

## 📌 Project Overview

This project demonstrates a complete **end-to-end Data Analytics workflow**:

1. 🌐 Web Scraping  
2. 🧹 Data Cleaning & Preprocessing  
3. 📊 Data Visualization & Insights Generation  

The objective of this project is to collect real-world data from the web, clean and preprocess it, and extract meaningful insights using visualization techniques.

# 🔹 Step 1: Web Scraping

📁 **File:** `1.project webscrapping.ipynb`

### 📌 Description
In this notebook, data is extracted from a website using web scraping techniques.

### 🛠️ Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

### 🔍 Process
- Sent HTTP requests to the target website  
- Parsed HTML content  
- Extracted required structured data  
- Converted data into a Pandas DataFrame  
- Saved the dataset for further processing  

### 🎯 Outcome
A structured dataset ready for cleaning and analysis.

# 🚀 How to Run This Project

1. Clone the repository:
   ```bash
   git clone <your-repository-link>
   ```

2. Navigate to project folder:
   ```bash
   cd project-folder
   ```

3. Install required libraries:
   ```bash
   pip install -r requirements.txt
   ```

4. Open Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

5. Run notebooks in order:
   - `1.project webscrapping.ipynb`
   - `2.Data_Ceaning_process.ipynb`
   - `3.Data_Vizualizations.ipynb`
